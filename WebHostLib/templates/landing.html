[EN] Welcome to Archipelago ... for Naix's Server!

This is a mirror web-host of the Archipelago framework for playing randomizer multiworld. This for was made by Naix for personal use on his Discord server. All credit goes to the original creators of Archipelago. Please refer to that site if you want to play outside Naix's server. The following is the introduction already found on the oficial site.

This is a cross-game modification system which randomizes different games, then uses the result to build a single unified multi-player game. Items from one game may be present in another, and you will need your fellow players to find items you need in their games to help you complete your own. For more details and answer to common questions please go to the F.A.Q. section.

This project is the cumulative effort of many talented people. Together, they have spent countless hours creating a huge repository of source code which has turned our crazy idea into a reality.
